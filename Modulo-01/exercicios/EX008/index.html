PRINCIPAIS FORMATAÇÕES DE TEXTO

NEGRITO EM DESTAQUE:

Nesta frase temos um Termo em Negrito usando a tag < B > que não é semântica.

Já nesta outra frase temos Este Termo em Negrito porem usando a tag STRONG que é semântica.

ITÁLICO / ÊNFASE:

Nesta frase temos um Termo em Itálico usando a tag < i > que não é semântica.

Já nesta outra frase temos Este Termo em Itálico porem usando a tag EM que é semântica.

TEXTO MARCADO:

Neste texto temos um trexo marcado usando a tag MARK e usando o CSS para dar cor, se nao usar CSS o texto fica marcado de amarelo como este.

TEXTO GRANDE E TEXTO PEQUENO:

Estamos criando um texto com um trexo grande com a tag BIG e um texto pequeno com a tag SMALL. OBS: O uso da tag BIG não é mais recomendado, está ficando absoleto aumentar a letra atraves do CSS.

TEXTO DELETADO:

Podemos marcar um texto como excluido, o usuario consegue ler ele porem deve desonciderar a leitura dele, usamos a tag DEL.

TEXTO INSERIDO:

também podemos marcar um texto como INSERIDO, ele fica sublinhado, usamos a tag INS elá é Semântica e tem tambem a tag U porem não recomendado por não ser semântica.

TEXTO ELEVADO / SOBRESCRITO :

Para inserir um texto do tipo X20 + 3 = ( usamos a tag sup )

TEXTO REBAIXADO / SUBSCRITO:
